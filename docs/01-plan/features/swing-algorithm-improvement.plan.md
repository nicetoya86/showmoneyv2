### 2.2 현재 알고리즘 기술신호 체계

```

  ✅ 일봉 정배열: SMA20 > SMA60

점수 체계 (상위):
  ✅ N자형 눌림목: +40점
  ✅ 컵핸들 패턴: +40점
  ✅ 20일 고점 돌파: +35점
  ✅ RVOL A등급 (3.0x): +25점
  ✅ 신고가 근접(95%): +25점
  ✅ OBV 수급 확인: +20점

AI 모델:
  ✅ PMAT (매수적정시점): 55~75% 기준
  ✅ 18개 입력 특징 (RSI, RVOL, ATR 등)
```

## 3. 갭 분석 (데이터 vs 현재 알고리즘)

### GAP-01: 금요일 진입 패널티 없음

- **데이터:** 금요일 평균 11.97% (전체 평균 대비 -0.62%)
- **현재:** 요일 구분 없음
- **개선안:** 금요일 진입 시 PMAT 기준 +5%p 상향 (더 엄격하게)

### GAP-02: 목표가 15% 상한 캡이 초과수익 차단

- **데이터:** 20%+ 수익 케이스가 15.9% (556건)
- **현재:** `CAP_TARGET_PCT = 0.15` (15% 고정 상한)
- **개선안:** 강한 모멘텀 조건(RVOL 3x+, ADX 30+, 점수 120+) 충족 시 캡을 20~25%로 확장

### GAP-03: 보유 기간 3일 고정 (유연화 필요)

- **데이터:** 58.9%가 1~2일, 21일+는 20.8% 평균
- **현재:** `holdingDays = 3` 고정
- **개선안:**
  - 기본 3일 유지 (현행)
  - 고강도 신호(점수 130+, RVOL 3x+) → 5일로 확장
  - 약한 신호(점수 80~100) → 2일로 단축

### GAP-04: 등급 개념 미반영

- **데이터:** "강매" 등급 30.34% 평균 수익률
- **현재:** 등급 개념 없음 (점수+AI 확률만 사용)
- **개선안:** 점수 130+ + PMAT 80%+ 조건을 "강매 등급"으로 별도 식별하여 Telegram 메시지에 표기

### GAP-05: 요일별 가중치 미적용

- **데이터:** 목요일 +0.57%, 금요일 -0.62% 차이
- **현재:** 요일 무관 동일 기준
- **개선안:** 요일별 rankScore 보정
  - 목요일: +3점 보너스
  - 수요일: +2점 보너스
  - 금요일: -5점 패널티
