# Plan: swing-benchmark-best

## Executive Summary

| 항목 | 내용 |
|------|------|
| Feature | swing-benchmark-best |
| 분석 기반 | `stock_best.xlsx` — 실제 수익 달성 스윙 종목 3,504건 (2018~2026) |
| 핵심 발견 | 수익 달성의 41.6%가 5~10% 구간에서 발생 / 매수 등급 실제 보유기간 5.64일 |
| 목표 | 데이터 기반 목표가·보유기간·등급 체계를 현재 알고리즘에 이식 |

### Value Delivered (4-Perspective)

| 관점 | 내용 |
|------|------|
| **Problem** | 현재 알고리즘의 목표가(ATR 2.8x ≈ 10~15%)는 달성 빈도가 낮고, 보유기간(3일)이 실제 수익 패턴(5.64일)보다 짧아 종목이 목표 도달 전에 만료됨 |
| **Solution** | 실수익 3,504건 분석 결과를 반영해 1차 목표가(+7%) 추가, 보유기간 연장, 매도차익 등급 신설 |
| **Function UX Effect** | 목표달성 건수 증가, 5~10% 수익 구간 캡처율 향상, 빠른 단기 수익 종목 별도 식별 |
| **Core Value** | 실증 수익 패턴을 알고리즘에 직접 이식 — 가설이 아닌 실데이터 기반 최적화 |

## 2. 현재 알고리즘과의 GAP 분석

### GAP-A: 보유기간이 실제 수익 패턴보다 짧음 (최고 우선순위)

| 항목 | 현재 | 데이터 | 차이 |
|------|------|--------|------|
| 매수 등급 보유기간 | 3일 | **5.64일** | **-2.64일 부족** |
| 강매 등급 보유기간 | 5일 | **38.66일** | **-33.66일 부족** |
| 완화 통과 보유기간 | 2일 | — | 합리적 |

> **직접 원인:** 이번 주 보유 7종목 (+4.7%~+24.2%)이 만료일 전에 목표 미달 상태. 3일이면 5.64일 평균 수익 패턴의 절반 수준에서 평가 종료됨.

### GAP-B: 5~10% 목표달성 구간 미설계

| 항목 | 현재 | 데이터 |
|------|------|--------|
| 목표가 | ATR 2.8x (≈10~15%) | 5~10% 구간이 41.6% |
| 1차 목표가 | 없음 | **없어서 이 구간 종목을 '보유중'으로 방치** |

> 5~10% 구간 종목이 목표가에 도달하지 못한 채 만료되거나 손절됨. 1차 목표 +7% 설정 시 41.6% 구간을 '목표달성'으로 캡처 가능.

### GAP-C: 매도차익 패턴 미식별 (단기 1.5일, 20%+ 평균)

| 항목 | 현재 | 데이터 |
|------|------|--------|
| 단기 급등 신호 | 없음 | 매도 등급 299건, 평균 20.26%, 1.5일 |
| 목표가 | ATR 2.8x | 더 낮은 목표로 빠른 청산 필요 |

> 갭상승 + 강한 수급 조합의 종목은 1~2일 안에 20% 수익 달성. 별도 식별 필요.

### GAP-D: 강매 등급 보유기간 고정 (5일 vs 실제 38.66일)

| 항목 | 현재 | 데이터 |
|------|------|--------|
| 강매 보유기간 | 5일 고정 | 38.66일 평균 |
| 트레일링 스탑 | 없음 | 장기 보유 시 필수 |

> 강매 등급은 5일이 아닌 목표가 도달까지 보유해야 함 (with 트레일링 스탑). 단, swing-profit-protection Plan에서 다루므로 연동.

## 4. 상세 개선안

### 4.1 HOLD-02 — 보유기간 연장

**변경 상수:**

```javascript
// 현재
const HOLD_STRONG = 5;   // 강매
const HOLD_NORMAL = 3;   // 매수
const HOLD_WEAK = 2;     // 완화

// 변경 후
const HOLD_STRONG = 0;   // 강매: 만료 없음 (트레일링 스탑으로만 관리)
const HOLD_NORMAL = 6;   // 매수: 3→6일 (데이터 5.64일 반올림)
const HOLD_WEAK = 2;     // 완화: 유지
```

> **HOLD_STRONG = 0**: 만료일을 설정하지 않고 swing-profit-protection의 트레일링 스탑이 청산 트리거 역할. weekly_reporter에서 0이면 '추적관리' 상태로 표기.

**기대 효과:**
- 이번 주 7개 보유 종목 중 상당수가 6일 이내에 목표 도달
- 강매 등급 종목의 30%+ 수익 기회 유지

### 4.2 TARGET-01 — 1차 목표가 신설 (+7%)

**개념:**
```
현재: 진입가 ─────────────────────► 목표가(ATR 2.8x ≈ 12%)
변경: 진입가 ──────► 1차목표(+7%) ──► 2차목표(ATR 2.8x ≈ 12%)
                        ↓
                   손절가 → 진입가+2%로 이동 (이익 잠금)
```

**변경 상수 추가:**
```javascript
const TARGET1_PCT = 0.07;    // 1차 목표: +7% (데이터 5~10% 구간 중앙값)
const TARGET1_STOP_MOVE = 0.02; // 1차 도달 시 손절 → 진입가+2%
```

**저장 데이터 변경 (weeklyRecommendations):**
```javascript
store.weeklyRecommendations[today2].push({
  ...기존 필드,
  target1: currentPrice * (1 + TARGET1_PCT),  // 1차 목표가 추가
  target: res.target,                          // 2차 목표가 (기존)
});
```

**Telegram 메시지 변경:**
```
- 1차목표: +7% (83,100원)    ← 신규
- 2차목표: +12.4% (89,500원) ← 기존 '목표가' → '2차목표'로 변경
- 손절가: -9.1% (72,800원)
```

### 4.3 SHORTTRADE-01 — 매도차익 등급 신설

**조건 (갭상승 + 강수급 조합):**
```javascript
// 매도차익 조건: 갭상승 + RVOL A등급 + RSI 상승세
const isShortTrade = (
  c.dailyChange >= 0.02 &&     // 당일 +2% 이상 갭
  rvolVal >= RVOL_GRADE_A &&   // RVOL A등급 (3x+)
  rsi >= 60 &&                 // RSI 60+
  score >= MIN_SCORE           // 기본 점수 충족
);
```

**등급 판정 추가:**
```javascript
const grade = isStrong ? '강매'
            : isShortTrade ? '매도차익'
            : strictPass ? '매수'
            : '관심';
```

**매도차익 등급 파라미터:**
```javascript
// 매도차익: 단기 빠른 청산
const HOLD_SHORTTRADE = 2;        // 2일 보유 (데이터 1.5일 기준)
const TARGET_SHORTTRADE_MULT = 1.5; // ATR 1.5x 목표 (빠른 달성)
```

**Telegram 메시지:**
```
[⚡단기] [스윙 포착] KOSDAQ | 종목명(코드)
등급: 매도차익 (단기2일)
- 목표가: +8.3% (ATR 1.5x)
- 손절가: -5.1% (ATR 1.0x)
```

### 4.4 REPORTER-02 — 주간 리포터 1차 목표 평가

**평가 로직 추가:**
```javascript
// target1 도달 여부 확인
if (!hitTarget1Day && r.target1 && high >= r.target1) {
  hitTarget1Day = localDate;
}

// 결과 판정: target1 도달도 '준달성'으로 분류
const result = hitTargetDay ? 'win'
             : hitTarget1Day ? 'partial_win'  // 1차 목표 달성
             : hitStopDay ? 'loss'
             : exitDate > today ? 'holding'
             : 'expired';
```

**리포트 표기:**
```
✅ 목표달성: N건 │ 🎯 1차달성: M건 │ ❌ 손절: K건
```
