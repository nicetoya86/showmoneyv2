# Plan: weekly-report-table-format

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 현재 주간 리포트는 종목당 2줄 텍스트 구조로 스크롤이 많이 필요하고 데이터 컬럼 간 구분이 불명확하여 가독성이 낮음 |
| **Solution** | 헤더 집계를 표 형식으로 정리하고, 종목별 정보를 1행 파이프(`│`) 구분자 형식으로 압축하여 표 형태로 제공 |
| **Function UX Effect** | 모바일 텔레그램에서 스크롤 없이 전체 주간 성과를 한눈에 파악 가능, 종목 비교가 용이 |
| **Core Value** | 주간 성과를 빠르게 스캔하여 투자 의사결정 효율성 향상 |

---

## 1. 현재 메시지 구조 (문제점)

### 1.1 현재 형식

```
[주간 스윙 성과 리포트]
📅 2026-03-23 ~ 2026-03-28
━━━━━━━━━━━━━━━━━━━━
총 추천: 5건  |  매수 진입: 4건
✅ 목표달성: 2건  ❌ 손절: 1건  🔄 보유: 1건
승률: 67%  (평가완료 3건)
━━━━━━━━━━━━━━━━━━━━

✅ 수익 종목
  삼성전자(005930)
    2026-03-25 | 목표달성 | 최고+8.3% (2026-03-26) | 115점

🔄 보유 종목
  에코프로비엠(247540)
    2026-03-24 | 보유중 | 최고+3.2% | 보유예정일2026-03-29 | 80점

❌ 손절 종목
  카카오(035720)
    2026-03-23 | 손절 | 최고+1.2% (2026-03-24) | 65점

⚪ 매수가 미도달 1건 (제외)
  LG화학(051910) | 2026-03-22
```

### 1.2 문제점 목록

| 문제 | 설명 |
|------|------|
| 종목당 2줄 | 종목명 줄 + 상세 줄로 분리되어 종목 수 × 2줄 스크롤 필요 |
| 집계 정보 혼잡 | 총추천/진입/목표/손절/보유가 한 줄에 뭉쳐 있어 파악이 어려움 |
| 컬럼 구분 불명확 | 각 항목이 `|` 구분자로는 있지만 들여쓰기가 섞여 있음 |
| 날짜 형식 장황 | `2026-03-25` 같은 전체 날짜가 반복되어 공간 낭비 |

## 2. 목표 메시지 구조

### 2.1 헤더 집계 (표 형식)

```
[주간 스윙 성과 리포트]
📅 2026-03-23 ~ 2026-03-28
━━━━━━━━━━━━━━━━━━━━
총추천  5건 │ 진입  4건 │ 승률 67%
✅ 목표 2건 │ ❌ 손절 1건 │ 🔄 보유 1건
━━━━━━━━━━━━━━━━━━━━
```

### 2.2 종목 섹션 (1행 1종목)

```
✅ 수익 (2건)
삼성전자(005930) │ 03-25 │ 최고 +8.3% │ 115점
카카오뱅크(323410) │ 03-26 │ 최고 +5.1% │ 90점

🔄 보유 (1건)
에코프로비엠(247540) │ 03-24 │ 최고 +3.2% │ 예정일 03-29

❌ 손절 (1건)
카카오(035720) │ 03-23 │ 최고 +1.2% │ 손절 03-24

⚪ 매수 미도달 (1건)
LG화학(051910) │ 03-22
```

### 2.3 푸터

```
⚠️ 본 리포트는 참고용이며 투자 결과에 책임지지 않습니다.
```

## 3. 상세 변경 사항

### 3.1 헤더 집계 섹션

| 항목 | 변경 전 | 변경 후 |
|------|--------|--------|
| 집계 표현 | `총 추천: 5건  \|  매수 진입: 4건` (1줄) + `✅ 목표달성: 2건  ❌ 손절: 1건  🔄 보유: 1건` (1줄) + `승률: 67%` (1줄) | `총추천 5건 │ 진입 4건 │ 승률 67%` (1줄) + `✅ 목표 2건 │ ❌ 손절 1건 │ 🔄 보유 1건` (1줄) |
| 날짜 구분자 | ` (평가완료 N건)` 별도 표기 | 승률 옆에 인라인으로 통합 |

### 3.2 수익 종목 섹션

| 항목 | 변경 전 | 변경 후 |
|------|--------|--------|
| 줄 수 | 종목명 1줄 + 상세 1줄 = 2줄/종목 | 1줄/종목 |
| 날짜 | `2026-03-25` (전체) | `03-25` (월-일만) |
| 형식 | `  종목명(코드)\n    날짜 │ 목표달성 │ 최고+X% (날짜) │ 점수` | `종목명(코드) │ 날짜 │ 최고 +X% │ 점수` |
| 섹션 헤더 | `✅ 수익 종목` | `✅ 수익 (N건)` |

### 3.3 보유 종목 섹션

| 항목 | 변경 전 | 변경 후 |
|------|--------|--------|
| 형식 | 2줄/종목 | 1줄/종목 |
| 보유예정일 | `보유예정일2026-03-29` | `예정일 03-29` |
| 기간만료 | `기간만료` 별도 표기 | `만료` (압축) |

### 3.4 손절 종목 섹션

| 항목 | 변경 전 | 변경 후 |
|------|--------|--------|
| 형식 | 2줄/종목 | 1줄/종목 |
| 손절일 | `손절 (2026-03-24)` | `손절 03-24` |

### 3.5 미진입 섹션

| 항목 | 변경 전 | 변경 후 |
|------|--------|--------|
| 형식 | `  종목명(코드) │ 날짜` | `종목명(코드) │ 날짜` (들여쓰기 제거) |
