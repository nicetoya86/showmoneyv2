QA MONITORING COMMANDS
================================================================================

View All Logs:
  tail -f logs/*.log

View Only Errors:
  tail -f logs/*.log | grep '"level":"ERROR"'

View Specific Stock:
  tail -f logs/*.log | grep 'STOCKCODE'

Parse JSON:
  cat logs/*.log | jq .

Group by Request ID:
  cat logs/*.log | jq -s 'group_by(.request_id) | .[]'
